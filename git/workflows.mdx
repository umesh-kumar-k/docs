### **1. Syncing (Git Remote) **[1]
**Remote Concept**: Bookmarks to repos (stored in `.git/config`)
```
git remote -v              # List: origin (fetch/push URLs)
git remote add john URL    # Add teammate repo
git remote rm john         # Remove
git remote rename old new  # Rename
```

**URLs**:
- **HTTP**: Read-only (`http://host/repo.git`)
- **SSH**: Read-write (`git@host:repo.git`)

**Subcommands**:
| Cmd | Effect |
|-----|--------|
| `show <name>` | Branches + push targets |
| `prune <name>` | Delete stale remote branches |
| `get-url <name>` | Fetch/push URLs |

### **3. Using Branches **[3]
**Branches = File Pointers** (`.git/refs/heads/`)
```
git branch feature        # Create
git checkout -b feature   # Create+switch
git checkout main         # Switch
git branch -d feature     # Safe delete (-D force)
git branch -a             # Local+remote (origin/...)
```

**Remote Sync**:
```
git push origin feature      # Create remote branch
git push origin :feature     # Delete remote branch
git push origin main         # Track (after -u)
```

**Detached HEAD**: `git checkout HEAD~2` (create branch before commit)
