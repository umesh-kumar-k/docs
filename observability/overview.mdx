Observability enables understanding complex systems via **MELT pillars** (Metrics, Events, Logs, Traces) + profiling, using OpenTelemetry (OTel) for vendor-neutral instrumentation across languages/environments, collected by agents → processed/exported to backends (New Relic/Datadog) for correlation/AIOps analysis.[newrelic][otel] Senior architects deploy OTel Collector (agent/gateway) + unified platforms for full-stack visibility, trading data volume/cost for proactive MTTR reduction (90%+ via traces).[1]

## Interview Summary
**MELT**: Metrics (Prometheus), Logs (ELK/Loki), Traces (Jaeger/Zipkin), Events (custom); **OTel**: APIs/SDKs for instrumentation → Collector (receive/process/export) → OTLP to backends; **Platforms**: New Relic/Datadog/Grafana (unified dashboards/AI); **AIOps**: Anomaly detection, root cause via correlation.[newrelic][2]

## Keywords & Patterns
- **Pillars**: Metrics (numerical KPIs), Events (state changes), Logs (structured/unstructured), Traces (spans/services).[newrelic]
- **OTel**: API/SDK (instrumentation), Collector (agent/gateway), OTLP (protocol), Semantic Conventions.[otel]
- **Tools**: Prometheus (metrics), Loki/Grafana (logs), Jaeger/Tempo (traces), New Relic/Datadog (unified).[1]
- **Processing**: Sampling (head/tail), Aggregation, Baggage (context), Propagation (W3C/B3).[3]
- **AIOps**: ML anomaly, Causal ML, SLO/SLI correlation.[newrelic]

## Common Trade-offs + Questions
OTel: Vendor-neutral/flexible vs config overhead; Unified platforms: Easy correlation vs cost/vendor lock; OSS (Prometheus): Free/custom vs enterprise support; High cardinality (custom metrics): Insight vs storage explosion.[newrelic][4]

**Example questions**:
- MELT correlation? (Traces link logs/metrics/events).[newrelic]

## Cheat-Sheet Q&A
**Q: Observability vs Monitoring?** A: Why (observability) vs What (monitoring).[newrelic]  
**Q: OTel Collector modes?** A: Agent (host), Gateway (central).[otel]  
**Q: OTLP?** A: Native protocol (all signals).[3]
**Q: Head vs Tail sampling?** A: Head (pre-collect), Tail (post-collect).[otel]  
**Q: High cardinality?** A: Exemplars, drop labels, aggregation.[4]
**Q: Baggage?** A: Cross-service context propagation.[otel]

**Traces**: Span trees (DAG), Flame graphs. **Metrics**: Timeseries (Prometheus TSDB), Downsampling. **Sampling**: Reservoir/probabilistic. **Correlation**: Trace ID propagation (W3C). **Anomaly**: Isolation Forest (AIOps).[newrelic][3]
