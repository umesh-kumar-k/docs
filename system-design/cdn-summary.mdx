## CDN Keywords & Terms (Skimmable Sheet)

### Core Components
- CDN (Content Delivery Network)  
- PoP (Point of Presence)  
- Edge server  
- Origin server  
- Cache hit / cache miss  

### Caching Mechanics
- TTL (Time to Live)  
- Cache-Control headers  
- ETag / Last-Modified  
- Cache warming  
- Cache invalidation  
- Cache purging  
- Stale-while-revalidate  
- Stale-if-error  
- Cache stampede  

### Routing & Network
- Anycast routing  
- DNS-based routing  
- GeoIP routing  
- BGP routing  
- HTTP/2, HTTP/3 (QUIC)  

### CDN Models
- Pull CDN  
- Push CDN  
- Reverse proxy CDN  
- Forward proxy CDN  

### Topologies
- Flat topology  
- Hierarchical topology  
- Mesh topology  
- Hybrid topology  
- Origin shield  

### Benefits & Metrics
- Latency reduction  
- Origin offload  
- Traffic spike absorption  
- DDoS mitigation  
- WAF (Web Application Firewall)  
- SSL/TLS termination  

### Policies & Rules
- Path-based caching  
- Vary headers  
- Cache key (query params, cookies, headers)  
- Cache bypass rules  
- Versioned URLs (cache busting)  

### Advanced Features
- Edge computing / serverless  
- Image optimization  
- Video streaming (HLS/DASH)  
- Bot management  
- Rate limiting at edge  

### Providers (Name-drop)
- Cloudflare, Akamai, CloudFront  
- Fastly, Google Cloud CDN  
- Azure CDN, KeyCDN  

### Use Cases
- Static assets (JS/CSS/images)  
- Media delivery (video/audio)  
- API caching (GET endpoints)  
- Global websites
